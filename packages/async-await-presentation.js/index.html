What is a Promise?

						
const p1 = Promise.resolve(5);
const p2 = Promise.reject(6);
const p3 = new Promise(resolve => {
	const user = 'my user' // db query or other async stuff
	resolve(user);
});
We can do pretty much everything with promises: - Store the resolve function for later - Attach thens to the promise (even though we haven't called resolve yet!) - Use setTimeout to wait for a while - Then call resolve, which will fulfill the promise and cause the thens to begin running new Promise constructor's callback is called synchronously see with catch, we can recover from a rejection, returning a fulfilled rather than rejected promise
